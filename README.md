# AI 기반 실시간 채팅/댓글 감정 분석 및 악성 댓글 탐지 프로그램
###### 감정 분석 딥러닝 모델을 활용하여 유튜브 댓글과 위버스 라이브 채팅에서 악성 표현을 탐지하고 관리

### ✨ 주요 기능
 * 댓글 및 실시간 채팅 수집
 * AI 기반 감정분석을 통한 악성 댓글/채팅 탐지
 * 악성 댓글/채팅 필터링
<br><br>
### 🪜 파이프라인
#### 1. YouTube 댓글 분석
 * __데이터 수집__: Selenium 이용해 YouTube 영상의 댓글을 스크롤링하며 수집
 * __DB 저장__: MySQL comments 테이블에 작성자, 댓글 내용, URL 정보 저장
 * __감정 분석__: DB에 저장된 댓글 내용을 딥러닝 모델 기반으로 긍정/부정 분류
   * 긴 댓글은 토큰 단위로 나눠 분석 후 최다 빈도 레이블로 결정
 * __결과 관리__: 레이블과 점수를 MySQL scores 테이블에 저장하고 원본 댓글과 합쳐 엑셀 파일로 출력

#### 2. Weverse 라이브 채팅 관리
* __실시간 모니터링__: Chrome 디버깅 모드와 Selenium 이용해 위버스에 접속하고 라이브 채팅을 실시간으로 읽어옴 
* __병렬 처리__: threading 사용해 채팅 영역 스크롤링과 감정 분석 및 제거 작업을 병럴 처리하여 실시간성 확보
* __자동 필터링__: 딥러닝 모델 기반으로 각 채팅을 감정 분석하고 악성(Negative)으로 판단될 경우 해당 채팅 요소를 DOM에서 바로 제거
<br><br>
### 🛠️ 기술 스택

| 분류 | 기술 | 설명 |
| :--- | :--- | :--- |
| **언어** | Python |
| **모델** | XLM-RoBERTa (Hugging Face) | 트위터에서 수집된 데이터로 학습 및 파인튜닝된 XLM-roBERTa 기반 다국어 감정 분석 모델 |
| **스크래핑/자동화**| Selenium, webdriver-manager  | 플랫폼 접속 및 댓글/채팅 수집 |
| **데이터베이스** | PyMySQL, SQLAlchemy | MySQL 데이터베이스 연결 및 데이터 입출력 관리 |
| **실시간** | threading | 위버스 라이브 채팅 실시간 스크롤링 및 분석 병렬 처리 |
